# ruby-eigen
Ruby binding for Eigen, a C++ template library for linear algebra, implemented using SWIG.

## How to install

### On Mac OS X

For Homebrew

    $ gem install rake-compiler
    $ gem install ruby-eigen

For MacPorts

    $ gem install rake-compiler
    $ gem install ruby-eigen

### On Ubuntu

    $ gem install rake-compiler
    $ gem install ruby-eigen
